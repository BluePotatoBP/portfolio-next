## Getting Started

Set your environmental variables:
```js
GITHUB_TOKEN=
CLOUDFLARE_SECRET_KEY=
NEXT_PUBLIC_TURNSTILE_KEY=
```

Run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000).
